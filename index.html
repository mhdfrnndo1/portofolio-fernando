About Me

Systems & Cloud Engineer

CLOUD & INFRASTRUCTURE
FULLSTACK DEVELOPMENT

Sebagai lulusan program studi Sistem Informasi dari Universitas Gunadarma, saya memadukan pemahaman fundamental arsitektur teknologi dengan eksekusi operasional tingkat enterprise. Perjalanan karier saya mencakup pengalaman komprehensif pada berbagai peran strategis di bidang Quality Assurance, manajemen aset IT, dan Cloud Engineering.

Pada aspek infrastruktur, saya terbiasa mengorkestrasi lingkungan data center dan mengelola ekosistem sistem operasi Linux secara optimal. Pengalaman ini mencakup penerapan dan peluncuran infrastruktur cloud publik maupun privat menggunakan perangkat seperti Virtuozzo, Proxmox, dan Sangfor. Prioritas operasional saya senantiasa berfokus pada pemeliharaan keandalan sistem, pemantauan performa jaringan secara proaktif, serta implementasi strategi disaster recovery yang terukur.

Melengkapi kompetensi infrastruktur tersebut, saya turut merancang solusi perangkat lunak yang presisi sebagai Fullstack Developer. Saya membangun antarmuka pengguna interaktif dan responsif memanfaatkan HTML, CSS, JavaScript, dan React.Js. Pada perancangan sistem logika bisnis, saya memanfaatkan ekosistem Node.Js, PHP, dan Python, serta menerapkan bahasa skrip LUA untuk mendukung kebutuhan automasi dan integrasi sistem tingkat lanjut.
