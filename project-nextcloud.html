Role Cloud Engineer

Project Overview

Di era kerja remote, memiliki kendali penuh atas data perusahaan adalah prioritas keamanan nomor satu. Proyek ini membebaskan perusahaan dari ketergantungan lisensi layanan Public Cloud pihak ketiga dengan membangun Private Data Center yang mandiri dan berdaulat penuh atas datanya (Data Sovereignty).

The Challenge

  • Biaya lisensi berlangganan (SaaS) per-user yang membengkak seiring pertumbuhan karyawan.
  • Risiko kebocoran dokumen rahasia jika disimpan di server asing.
  • Kebutuhan mutlak akan fitur live-editing dokumen (Excel, Word) oleh banyak divisi secara bersamaan.

Technical Execution

Arsitektur sistem dibangun di atas Docker Containers untuk isolasi servis yang ketat antara Nextcloud Web App dan Collabora CODE Server. Performa baca/tulis dokumen ditingkatkan drastis menggunakan in-memory cache Redis dan optimasi MariaDB. Trafik jaringan dilindungi Reverse Proxy dengan manajemen sertifikat SSL otomatis.
